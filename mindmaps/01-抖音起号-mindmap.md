# 小黑托昵抖音起号 Mindmap

```mermaid
mindmap
  root((小黑托昵抖音起号执行手册))
    1. 账号定位与人设
      目标：同城到店成交
      人设：老板娘替宝妈筛选儿童贴身棉品
      产品线：内裤 / 背心 / 秋衣秋裤 / 家居服 / 袜品
      记忆顺序：先记住人，再记住门店
      红线：不做泛童装号 / 不做官方宣传号 / 不做低价号
    2. 首页配置与置顶内容
      昵称：[城市]小黑托昵老板娘
      简介：帮宝妈省选面料和尺码时间
      置顶 1：我是谁
      置顶 2：换季怎么选
      置顶 3：来店能解决什么
      视觉：门店现场、货架、摸面料、试穿区
    3. 内容系统与栏目比例
      50% 真实门店日常
      30% 宝妈决策内容
      20% 到店证明 / 轻转化
      产品优先级：秋衣秋裤 -> 内裤 -> 家居服 -> 背心 -> 袜品
      固定结构：问题 -> 现场 -> 结论 -> CTA
      题型池：接待、换季、问答、尺码、复购、上新、试穿、直播切片
    4. 14 天执行排期
      第 1 周：覆盖接待 / 换季 / 问答
      第 2 周：放大优胜题型
      默认放大：换季选品 + 尺码判断
      每日节奏：拍摄 -> 发布 -> 回评
      直播：Day 6 / Day 13 19:30
    5. 短视频脚本模板与开场库
      模板 1：真实接待
      模板 2：换季选品
      模板 3：尺码判断
      模板 4：问题答复
      模板 5：老客复购 / 到店证明
      开场库：15 条问题钩子
    6. 直播 SOP
      定位：答疑 + 选码 + 信任
      禁忌：不硬卖、不喊价
      流程：开场 -> 集中答疑 -> 实物演示 -> 同城承接
      直播后：剪切片 + 补选题
    7. 评论区 / 私信 / 到店转化 SOP
      三类高价值问题：地址 / 尺码 / 最近适不适合来
      评论区标准回复
      私信四步：年龄 -> 场景 -> 缩范围 -> 引导到店
      到店第一句：你是从抖音来的吗
      每日记录：地址咨询 / 尺码咨询 / 到店提及抖音
    8. 风险避雷与合规话术
      禁用：最安全 / 绝对放心 / 全网最好
      替代：这个场景更适合 / 反馈更稳定
      边界：不做品牌官方背书
      敏感问题：引导回官方公开信息和门店经验
      拍摄边界：顾客和儿童隐私保护
    9. AI 工具完整工作流
      环节 1：选题与脚本
        Kimi
        豆包
        腾讯元宝
        最推荐：Kimi
      环节 2：画面灵感与封面
        即梦 AI
        剪映 AI 图片
        豆包 AI 写真 / 分身
        最推荐：即梦 AI
      环节 3：剪辑与字幕
        剪映手机版
        剪映桌面版
        剪映网页版
        最推荐：剪映手机版
      环节 4：回复与复盘
        Kimi Sheets / Docs
        豆包 + 云盘
        腾讯元宝
        最推荐：Kimi Sheets / Docs
    10. 每周复盘表与淘汰规则
      只看 4 个指标
        同城播放占比
        宝妈咨询数
        到店提及抖音人数
        哪类开场最留人
      保留：有强购买问题
      观察：有播放没咨询
      淘汰：连续 2 条空流量
      下周：连发 3-5 条优胜系列
```
