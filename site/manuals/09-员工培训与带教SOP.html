小黑托昵员工培训与带教SOP¶

版本：v1.0
更新日期：2026-03-12
适用对象：门店新员工、转岗员工、需要统一服务标准的老员工
配套文档： - docs/source/06-店铺经营基础框架.md - docs/source/08-小团队门店分工与排班手册.md

使用说明¶

  • 这份文档解决的是 1 个老板怎么带 2 位员工，新人怎么上手，老员工怎么校准。
  • 培训目标不是让员工背很多概念，而是让员工能：
  • 接住顾客
  • 说清怎么选
  • 看懂门店高频问题
  • 跟上抖音引流带来的顾客问题

1. 培训目标¶

员工培训先确保这 4 件事达标：

  1. 知道门店主卖什么、怎么帮顾客选
  2. 知道接待顺序，不乱推货
  3. 知道抖音来的顾客该怎么接
  4. 知道每天哪些信息必须反馈给老板

2. 新员工 7 天上手路径¶

Day 1：认识门店和基本规则¶

要学会：

  1. 门店区域和货架分布
  2. 基本岗位纪律
  3. 开店 / 闭店基本动作
  4. 顾客进店后的基础礼貌和接待方式

当天过关标准：

  • 能说清店里主要区域
  • 知道谁负责什么
  • 不会在顾客面前发懵

Day 2：认识产品逻辑¶

要学会：

  1. 先按厚薄分类
  2. 再按年龄 / 尺码缩小范围
  3. 知道常见顾客纠结点

当天过关标准：

  • 能把顾客带到大致正确区域

Day 3：接待顺序训练¶

要学会：

  1. 先问需求
  2. 再给范围
  3. 再做推荐
  4. 不一上来堆很多款

当天过关标准：

  • 能完成 1 次基础接待，不严重跑偏

Day 4：抖音到店顾客承接¶

要学会：

  1. 听懂顾客是不是线上看过来的
  2. 接上顾客在线上问过的问题
  3. 及时把关键顾客情况告诉老板

当天过关标准：

  • 能识别并配合接住 1 位抖音到店顾客

Day 5：尺码与常见问题训练¶

要学会：

  1. 常见尺码判断逻辑
  2. 常见年龄段问题
  3. 常见厚薄问题

当天过关标准：

  • 遇到常见问题时知道先怎么回应

Day 6：老客与复购训练¶

要学会：

  1. 识别老客
  2. 记录老客偏好
  3. 不把老客当普通路过顾客处理

当天过关标准：

  • 能说出 1 位老客的典型购买逻辑

Day 7：综合带教考核¶

要学会：

  1. 独立完成基础接待
  2. 配合收尾和记录
  3. 说出本周最常被问的 3 个问题

当天过关标准：

  • 老板确认可以进入正常排班

3. 产品知识培训怎么带¶

不要从复杂品牌故事开始，先按顾客最关心的顺序教：

  1. 什么时候该换
  2. 厚薄怎么选
  3. 不同年龄段怎么挑
  4. 尺码怎么判断
  5. 顾客最容易纠结什么

3.1 当前产品线先教哪 5 类¶

员工培训先统一这 5 类产品，不要求一开始全讲很深，但至少知道各自先看什么：

  1. 内裤 先看尺码、贴身感、孩子穿着反馈
  2. 背心 / 发育期近身打底 先看年龄段、过渡期需求、基础尺码
  3. 秋衣秋裤 / 单裤 / 打底 先看厚薄、换季场景、活动量
  4. 家居服 / 睡衣 先看场景、体感、松量
  5. 袜品 / 连裤袜 先看搭配场景和基础尺码，不要讲得太复杂

3.2 员工最少要会说的产品逻辑¶

员工不用把每一类都讲成长篇，只要先能说出：

  1. 这类主要帮顾客解决什么问题
  2. 这类先看厚薄、尺码还是场景
  3. 这类什么时候要请老板来接

培训方法：

  1. 看货架讲一遍
  2. 拿实物讲一遍
  3. 让员工自己复述一遍
  4. 接待后再复盘一遍

4. 接待话术怎么练¶

4.1 基础接待开口¶

统一先练这句：

是自己先看看，还是我先按厚薄和尺码帮你缩一下？

4.2 顾客犹豫时¶

统一先练这句：

这种情况我一般先帮宝妈把范围缩小，不然会越看越乱。

4.3 顾客提到刷到抖音时¶

统一先练这句：

那我先按你线上关心的问题带你看，不重新让你从头选。

4.4 五类产品的开口重点¶

  1. 内裤 你家孩子现在主要是卡在尺码，还是穿着舒服度这一步？
  2. 背心 你是想看日常打底，还是最近这个阶段更需要近身过渡的？
  3. 秋衣秋裤 / 打底 你现在是准备最近穿，还是给后面换季先备着？
  4. 家居服 你更在意在家穿着轻松一点，还是想兼顾早晚温差？
  5. 袜品 你是想顺手配一类，还是孩子最近正好缺这类？

5. 培训时最该纠正的 5 个问题¶

  1. 一上来就推款，不先问需求
  2. 一次说太多，让顾客更乱
  3. 顾客说看过抖音，员工却接不上
  4. 只顾整理货，不顾正在看货的顾客
  5. 接待完没有留下任何复盘信息
  6. 不分产品类别，所有顾客都用同一套说法

6. 培训考核怎么做¶

6.1 每天看 3 件事¶

  1. 接待是否自然
  2. 推荐顺序是否正确
  3. 是否会记录问题

6.2 每周看 3 件事¶

  1. 是否能独立接常见顾客
  2. 是否能识别抖音到店顾客
  3. 是否能把高频问题反馈回来

7. 老板怎么带 2 位员工¶

小团队门店里，带教核心还是老板亲自带。不要只说“你多看看”，而要做到：

  1. 先示范一次
  2. 再让员工做一次
  3. 当天就指出 1-2 个关键问题
  4. 第二天继续看是否改掉

如果有两位员工，建议这样带：

  1. 先把接待最稳的员工带成 协助带教的人
  2. 新员工优先跟着老板或熟手一起接待
  3. 不让两个不熟练的人同时顶高峰时段

8. 当前最先执行的培训动作¶

如果你现在人手不多，先只做这 4 项：

  1. 新员工入店前 7 天带教表
  2. 基础接待开口统一
  3. 抖音到店顾客识别训练
  4. 每天关店前 5 分钟培训复盘

只要这 4 项开始跑，门店培训就不是“讲过了”，而是“真的开始形成标准”。
