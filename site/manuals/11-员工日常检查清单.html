小黑托昵员工日常检查清单¶

版本：v1.0
更新日期：2026-03-12
适用对象：1 个老板 / 老板娘 + 2 位员工的小团队门店
配套文档： - docs/source/08-小团队门店分工与排班手册.md - docs/source/09-员工培训与带教SOP.md - docs/source/10-老板周复盘与执行校准.md

使用说明¶

  • 这份文档解决的是 员工每天到底要检查什么，做到什么才算到位。
  • 目标不是增加负担，而是让两位员工每天都有固定动作，不靠临时提醒。
  • 建议用法：
  • 开店前看一次
  • 中途空档快速过一次
  • 闭店前再对一次

1. 每天最先检查的 5 件事¶

员工每天上岗后，先确认这 5 件事：

  1. 今天自己是 主接待 还是 辅助补位
  2. 今天老板是否在店，哪些顾客要优先交给老板
  3. 今天是否有拍摄 / 直播 / 活动安排
  4. 重点货架和尺码是否齐
  5. 昨天交接的问题今天是否还要继续跟

2. 开店前检查清单¶

2.1 门店状态¶

  1. 门口、收银台、试穿区是否整洁
  2. 重点货架是否整齐
  3. 常问尺码是否好拿、好找
  4. 店内是否有明显缺货或乱放

2.2 自己的工作状态¶

  1. 知道今天谁主接待
  2. 知道今天谁主整理补位
  3. 知道今天如果遇到抖音顾客先怎么接
  4. 知道今天老板特别关注哪类顾客或哪类货

3. 营业中检查清单¶

3.1 接待检查¶

员工营业中至少要反复检查这 5 件事：

  1. 顾客进店后有没有被及时招呼
  2. 自己有没有先问需求，再推荐
  3. 顾客一旦提到抖音，有没有及时接上线上问题
  4. 顾客犹豫时，有没有先帮她缩小范围
  5. 关键顾客有没有及时交给老板

3.2 现场补位检查¶

  1. 另一位同事接待时，自己有没有及时补位整理
  2. 货架乱了有没有及时顺手整理
  3. 试穿区是否保持可用
  4. 空档时有没有及时补货、归位

4. 抖音来客专项检查¶

只要遇到抖音来的顾客，员工先检查这 4 件事：

  1. 有没有听清顾客是从视频、直播还是评论区过来的
  2. 有没有听清她最关心的是 地址 / 尺码 / 最近适不适合来 / 厚薄
  3. 有没有把关键问题及时告诉老板
  4. 接待完后有没有把高频问题记下来

5. 闭店前检查清单¶

5.1 现场收尾¶

  1. 货架是否恢复整齐
  2. 试穿区是否整理完成
  3. 收银台和门口区域是否清爽
  4. 明天高频会用到的货是否已归位

5.2 信息记录¶

  1. 今天有没有抖音到店顾客
  2. 今天最常被问的 1-3 个问题是什么
  3. 今天有没有顾客需要明天继续跟
  4. 今天哪类顾客最容易成交

6. 交接班检查清单¶

交接班时，员工至少讲清这 6 件事：

  1. 现在店里还有没有正在看的顾客
  2. 今天有没有抖音来的顾客
  3. 哪类问题今天被反复问到
  4. 哪些尺码 / 货架需要补
  5. 有没有需要特别注意的顾客
  6. 有没有老板交代必须继续跟的事

7. 员工每天最容易漏掉的 5 个点¶

  1. 顾客提到抖音，却没有继续追问她具体看了什么
  2. 顾客看起来犹豫，却只重复介绍，不帮她缩小范围
  3. 接待结束后没记问题，第二天又从头来
  4. 只顾整理货架，没注意顾客已经进店
  5. 知道有问题，但没有及时告诉老板

8. 每天最小执行标准¶

如果你不想一开始做太复杂，先要求员工每天做到这 4 条：

  1. 知道自己当天的主职责
  2. 知道抖音顾客怎么识别和交接
  3. 闭店前说出今天最常被问的 1 个问题
  4. 交接班时不漏重点顾客和重点问题

9. 当前最建议的执行方式¶

最适合先落地的方法不是做复杂表格，而是：

  1. 把这份清单打印或贴在员工能看到的位置
  2. 开店前老板口头过 1 次重点
  3. 闭店前让员工自己复述今天有没有做到

先把清单跑顺，再决定要不要把它转成打卡表。
