## 命名规则

- 主文档统一放在 `docs/`
- 脑图统一放在 `mindmaps/`
- 文件名前缀用两位序号，保证长期可排序
- 同主题的 Markdown 和脑图尽量同名

## 更新规则

- 改策略、SOP、脚本、学习路径时，优先改 `docs/`
- 改了章节结构后，同步更新 `mindmaps/`
- 新增专题时，沿用：
  - `docs/04-主题名.md`
  - `mindmaps/04-主题名-mindmap.md` 或 `.xmind`
