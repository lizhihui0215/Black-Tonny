# 小黑托昵库存预警与补货计算手册

版本：v1.0  
更新日期：2026-03-13  
适用对象：1 个老板 / 老板娘 + 2 位员工的小团队门店  
配套文档：
- `docs/source/07-店铺日常经营复盘表.md`
- `docs/source/15-门店营业额增长框架.md`
- `docs/source/18-门店库存与营收核心计算方法.md`

## 使用说明

- 这份文档解决的是 `什么时候该补货、补多少、哪些货已经开始危险`。
- 目标不是做复杂 ERP，而是让门店先有一套能用的库存预警规则。
- 默认优先管理 `秋衣秋裤 / 打底、内裤、家居服、背心、袜品` 这 5 类。

## 1. 库存管理的目标

先把库存管理理解成同时解决这 4 件事：

1. 不断码
2. 不压太多货
3. 不靠感觉补货
4. 不让慢货吃掉营业额空间

## 2. 先统一库存记录口径

### 2.1 门店最小记录单位

对小团队门店，先不用一上来管太复杂。  
建议按这个层级记：

1. `品类`
2. `款号 / 系列`
3. `尺码`
4. `当前在手件数`
5. `在途件数`

### 2.2 先重点盯住什么

不是所有货都要一样细盯，先盯：

1. 高频主销款
2. 高频尺码
3. 季节切换款
4. 最近 14 天卖得最快的款

## 3. 先记住 4 个关键数字

### 3.1 当前可售库存

`当前可售库存 = 在手库存 - 已预留 / 不可售件数`

### 3.2 近 14 天日均销量

`近14天日均销量 = 近14天销售件数 ÷ 14`

### 3.3 库存覆盖天数

`库存覆盖天数 = 当前可售库存 ÷ 近14天日均销量`

这条最关键，因为它直接告诉你：

- 还能卖几天
- 是不是快断了
- 是不是压太多了

### 3.4 补货提前期

`补货提前期` 指的是从你下单到货真正能上架卖，需要多少天。

对门店来说，这个数字必须记清楚，因为补货晚不晚，主要就卡在这里。

## 4. 一套门店能直接执行的预警线

### 4.1 按覆盖天数看预警

| 状态 | 库存覆盖天数 | 说明 | 动作 |
| --- | --- | --- | --- |
| 红灯 | `<= 3天` | 很容易断货 | 当天处理 |
| 黄灯 | `4-7天` | 需要准备补货 | 24 小时内确认 |
| 绿灯 | `8-21天` | 相对健康 | 继续观察 |
| 黑灯 | `> 45天` | 压货风险高 | 暂停补货、考虑去化 |

### 4.2 哪些情况直接算红灯

下面 3 种情况，不用等算式，直接按红灯处理：

1. 主销尺码已经缺码
2. 周末前库存明显不够
3. 商场活动前核心款不齐

## 5. 安全库存和补货点怎么算

### 5.1 安全库存

最适合小店的简化算法：

`安全库存 = 近14天日均销量 × 安全天数`

建议先这样设：

| 品类 | 建议安全天数 |
| --- | --- |
| 秋衣秋裤 / 打底 | `7-10天` |
| 内裤 | `7-10天` |
| 家居服 | `5-7天` |
| 背心 | `5-7天` |
| 袜品 | `3-5天` |

### 5.2 补货点

`补货点 = 近14天日均销量 × 补货提前期 + 安全库存`

解释：

- 前半段保证你等货时不断
- 后半段给你留一点缓冲

### 5.3 建议补货量

`建议补货量 = 目标库存 - 当前可售库存 - 在途件数`

其中：

`目标库存 = 近14天日均销量 × 目标覆盖天数 + 安全库存`

建议先用这组目标覆盖天数：

| 品类 | 目标覆盖天数 |
| --- | --- |
| 秋衣秋裤 / 打底 | `10-14天` |
| 内裤 | `10-14天` |
| 家居服 | `7-10天` |
| 背心 | `7-10天` |
| 袜品 | `5-7天` |

## 6. 一个门店能直接算的例子

假设某款秋衣秋裤 M 码：

1. 近 14 天卖了 `28 件`
2. 日均销量 = `28 ÷ 14 = 2`
3. 补货提前期 = `4 天`
4. 安全天数 = `8 天`
5. 当前可售库存 = `18 件`
6. 在途件数 = `0 件`

那么：

1. `安全库存 = 2 × 8 = 16`
2. `补货点 = 2 × 4 + 16 = 24`
3. 当前库存 `18 < 24`，已经该补
4. 如果目标覆盖天数按 `14 天`
5. `目标库存 = 2 × 14 + 16 = 44`
6. `建议补货量 = 44 - 18 - 0 = 26`

这个结果的意思不是必须一口气补 26 件，而是告诉你：

- 这款已经进入补货区间
- 需要结合预算和季节节点，决定分几次补

## 7. 每天 / 每周怎么执行

### 7.1 每天

每天至少看这 3 件事：

1. 有没有红灯 SKU
2. 有没有主销尺码快断
3. 今天卖得最快的款是不是要提前补

### 7.2 每周

每周固定做这 4 件事：

1. 看一遍主销款覆盖天数
2. 看一遍黑灯慢货
3. 定本周补货单
4. 定哪些货先不再进

## 8. 1 个老板 + 2 位员工怎么分工

### 8.1 老板 / 老板娘

主要负责：

1. 看红黄黑灯
2. 拍板补货和停补
3. 判断季节和活动节点

### 8.2 员工 A

主要负责：

1. 日常点主销尺码
2. 提醒红灯款
3. 记录当天卖得最快的类目

### 8.3 员工 B

主要负责：

1. 记录在手和在途
2. 盯陈列和补位
3. 标记慢货和缺码

## 9. 当前最先执行的 5 件事

如果你今天就开始管库存，先做这 5 件：

1. 给主销款和主销尺码建一张表
2. 每天记 `当前可售库存`
3. 每周算一次 `库存覆盖天数`
4. 用 `红灯 / 黄灯 / 绿灯 / 黑灯` 做最小预警
5. 补货时先按公式算，再结合预算决定深浅

先把这 5 条跑顺，库存管理就会从“凭感觉”变成“有边界”。
