# 全量历史数据导出与看板升级清单

版本：v1.0  
更新日期：2026-03-13  
适用对象：老板 / 老板娘、数据协作人员

## 这份文档解决什么问题

这份文档解决的是：

1. 为什么当前库存销售看板只显示最近几天的销售趋势
2. 如果要看到 `2025-03` 到现在的完整趋势，应该从系统里再导什么
3. 数据补齐以后，看板会升级成什么样

## 当前这包数据为什么只显示最近 8 天趋势

当前看板里的日销趋势，来自：

- `销售清单 - 【2026-03-13】.xlsx`

这张表里实际带出来的销售日期范围是：

- `2026-03-06` 到 `2026-03-13`

所以程序没有办法凭空还原 `2025-03` 到现在的逐日销售趋势。  
它现在能做的是：

1. 用 `销售清单` 看最近 8 天的运营情况
2. 用 `商品销售情况` 看历史累计结果

## 当前这包数据已经包含的历史信息

虽然销售明细只有最近 8 天，但压缩包里已经有一部分历史累计信息：

1. `商品销售情况【2026-03-13】.xls`
   - 最早首次到货日期：`2025-03-17`
   - 最早首次销售日期：`2025-03-17`
   - 可看历史累计销售额、累计销量、总到货、当前库存
2. `出入库单据【2026-03-13】.xlsx`
   - 时间范围约 `2025-03-18` 到 `2026-03-12`

所以：

- `短期趋势` 已经能看
- `历史累计` 已经能看
- `完整逐日/逐月趋势` 还缺全量销售明细

## 最推荐的补导方式

### 目标时间范围

建议直接导：

- `2025-03-01` 到 `2026-03-13`

如果系统限制一次不能导这么长，也至少按月补导，直到补齐这段时间。

### 必导表

最少导这 4 张：

1. `销售清单`
   - 时间范围：`2025-03-01` 到 `2026-03-13`
   - 作用：做逐日、逐周、逐月销售趋势
2. `零售明细统计`
   - 时间范围：同上
   - 作用：做品类、款号、尺码层级分析
3. `商品销售情况`
   - 时间范围：导最新
   - 作用：保留累计口径、售罄、周销量
4. `出入库单据`
   - 时间范围：`2025-03-01` 到 `2026-03-13`
   - 作用：看补货、调拨、入库节奏

### 选导表

如果系统支持，建议一起补：

1. `会员综合分析`
   - 作用：做会员增长、复购和高价值会员分析
2. `导购员报表`
   - 作用：做导购月度表现趋势
3. `库存零售统计`
   - 作用：做品类存销比对

## 如果系统不支持一次导一年

优先这样处理：

1. 每个月导一份 `销售清单`
2. 文件名保留月份，例如：
   - `销售清单_2025-03.xlsx`
   - `销售清单_2025-04.xlsx`
3. 最后统一放进一个文件夹或一个 zip

程序后面可以改成自动合并多个月份文件。

## 导出时一定要保留的字段

### 销售清单

必须保留：

1. `销售日期`
2. `店铺名称`
3. `零售单号`
4. `商品大类`
5. `款号`
6. `颜色`
7. `尺码`
8. `数量`
9. `金额`
10. `会员卡号`
11. `服务导购`

### 出入库单据

必须保留：

1. `发货时间`
2. `接收时间`
3. `接收店铺`
4. `商品款号`
5. `颜色`
6. `尺码`
7. `数量`
8. `单据类型`

## 数据补齐后，看板会升级什么

补齐全量历史销售明细后，下一版最值得加这 6 块：

1. `2025-03 至今月度销售趋势`
2. `月度品类销售结构`
3. `月度会员销售占比`
4. `月度导购表现趋势`
5. `月度库存覆盖变化`
6. `断码与补货节奏变化`

## 最推荐的执行顺序

1. 先补导 `销售清单` 全量历史
2. 再补导 `出入库单据` 全量历史
3. 保留最新的 `商品销售情况`
4. 把文件打成一个 zip 发给我
5. 我再把看板升级成完整历史趋势版

## 当前版本你先怎么看

在全量历史明细补齐之前，当前看板建议这样理解：

1. `最近 8 天趋势` 用来做日常运营判断
2. `历史累计销售额 / 累计销量` 用来判断门店从开店至今的大盘
3. 不要把最近 8 天误当成开店以来全量趋势
