# 小黑托昵小团队门店分工与排班手册

版本：v1.0  
更新日期：2026-03-12  
适用对象：1 个老板 / 老板娘 + 2 位员工的小团队门店  
配套文档：
- `docs/source/06-店铺经营基础框架.md`
- `docs/source/07-店铺日常经营复盘表.md`

## 使用说明

- 这份文档解决的是 `3 个人怎么分工、怎么排班、现场怎么不断档`。
- 默认先做最小化管理，不上来做复杂制度。
- 目标是让小团队门店也能做到：
  - 接待不断层
  - 顾客问题有人接
  - 抖音到店顾客不被漏掉
  - 老板和两位员工都知道自己该做什么

## 1. 管理目标

小团队门店先盯住这 4 件事：

1. 哪怕只有 3 个人，也不能让门店现场乱
2. 老板负责关键成交，两位员工负责基础接待、补位和整理
3. 抖音来的顾客能被识别并顺利接住
4. 交接班和当日重点说清楚，不靠临时猜

## 2. 角色分工

### 2.1 老板 / 老板娘

核心职责：

1. 定门店方向和服务标准
2. 负责关键顾客、关键成交和关键问题拍板
3. 负责把抖音内容和门店经营打通
4. 每周看复盘，决定下周重点

### 2.2 员工

核心职责：

1. 完成基础接待、整理、补货、收尾
2. 按统一话术先帮顾客缩小选择范围
3. 识别抖音到店顾客并及时告诉老板
4. 记录高频问题，不把有价值信息丢掉

说明：

- 两位员工不一定要分成固定头衔，但要明确当天谁是 `主接待`，谁是 `辅助接待 / 整理补位`。

## 3. 每人每天必须完成的事

### 3.1 老板 / 老板娘每日动作

1. 看当天排班是否合理
2. 处理 1-2 个关键顾客、高价值咨询或老客复购
3. 抽查 1 次员工接待是否符合统一话术
4. 关店前看当天复盘结论

### 3.2 员工每日动作

1. 按要求完成开店准备
2. 接待顾客时先判断问题，再带货
3. 记录抖音来的顾客和高频问题
4. 完成闭店前整理和当日交接

## 4. 排班原则

### 4.1 最小排班逻辑

先按这 4 条排：

1. 高峰时段尽量 `老板 + 1 位员工` 同时在店
2. 周末、换季、活动日尽量 `3 人有重叠时段`
3. 拍视频、直播时，必须留 1 个人专门顾现场
4. 新员工尽量不要单独顶高峰时段

### 4.2 小团队排班建议

建议先按这 4 种状态理解，不一定要排成复杂班表：

1. `老板单人在店`
   适合处理关键顾客、老客承接和高价值成交
2. `老板 + 1 位员工`
   适合大多数正常营业时段
3. `2 位员工在店`
   适合老板短时外出，但要避免放在高价值成交时段
4. `3 人重叠在店`
   适合周末、高峰、换季、拍摄、直播、活动日

### 4.3 排班时必须考虑的 5 个因素

1. 哪几天顾客最多
2. 哪些时段更容易出现抖音到店
3. 新人是否需要带教
4. 是否安排了拍摄 / 直播
5. 哪些时段必须由老板亲自接关键顾客

## 5. 交接班要求

三个人轮换也不能只说“差不多”，至少要交这 6 件事：

1. 店里还有没有正在看的顾客
2. 今天有没有抖音来的顾客
3. 哪类问题今天被反复问到
4. 哪些货架或尺码需要补
5. 今天是否有老客需要后续跟进
6. 有没有必须提醒下一位同事注意的顾客情况

## 6. 开店 / 闭店清单

### 6.1 开店前

1. 检查门店整洁度和试穿区状态
2. 检查重点货架和尺码是否齐
3. 确认今天谁主接待、谁主整理补位、谁盯收银
4. 看一眼当天是否有内容拍摄或直播安排

### 6.2 闭店前

1. 整理货架和试穿区
2. 记录当天抖音到店和高频问题
3. 记录老客复购和高价值咨询
4. 写当天 1 句复盘

## 7. 小团队门店最重要的纪律

先只抓最影响顾客体验的 6 条：

1. 顾客进店后不能长时间无人招呼
2. 不在顾客面前互相推责任
3. 不随意改口径，线上线下话术尽量一致
4. 不把顾客问题记在脑子里不落记录
5. 不因拍视频影响门店基本接待
6. 不把“没空”当成不执行复盘的理由

## 8. 请假 / 调班最小规则

### 8.1 请假

1. 尽量提前报备
2. 说明影响的是哪个班次
3. 明确谁来顶班，不能只抛问题

### 8.2 调班

1. 先确认门店是否还有人能接住基础接待
2. 调班后重新确认谁负责主接待、谁负责补位
3. 老板必须知道最终排班结果

## 9. 当前最适合门店先执行的 3 件事

如果你现在还没有成熟管理系统，先只执行：

1. 每天明确谁负责 `接待主位`
2. 每天交接班时固定讲 `今天有没有抖音来的顾客`
3. 每周固定看一次 `谁最能接住顾客问题`

先把这 3 件事做稳，小团队门店管理就会明显比之前顺。
