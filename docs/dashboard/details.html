经营策略

这里的策略会根据每日数据、整体数据、库存和利润状态自动调整。

自动生成经营方案

基于当前数据生成的处理方案，适合老板拍板。

库存周期过长，先做促销去化
触发原因：当前经营库存覆盖天数 303.1 天，已经明显偏长。
目标：先把高库存慢销货动起来，给当季主销和补货腾出空间。
袜品 组合促销会员定向去化促销周末限时清货台
  • 袜品 组合促销：围绕 袜品 做 2 件 / 3 件组合价，优先配基础内裤、袜品、家居这类低决策商品，门店主推“顺手带走”的组合。
  • 会员定向去化促销：优先触达 圆子、茵茵、呦米 这一类老客，发换季组合包或加价购，主推“到店试一套 / 顺手补一轮”。
  • 周末限时清货台：把 袜品 和当前去化重点放到门口清货位，做 3 天限时陈列，配合第二件折扣或满额换购，但不全场乱打折。
畅销款缺货风险高，补货要分级
触发原因：当前建议补货 SKU 150 个，补货压力较高。
目标：先保住最能带营业额的款，不让补货动作失控。
A类马上补B类本周补C类先观察
  • A类马上补：今天先定 裤类 这个品类的补货预算，再下钻到销售金额高、库存 0-1、库存周数小于 1 的款。
  • B类本周补：本周内处理同品类里销售稳定但库存还有 1-2 周的款，避免一下子把补货预算打满。
  • C类先观察：春季 和下季交界的款先小量试补，避免把非主销季的货压深。
负库存先纠偏，再谈补货
触发原因：当前负库存 SKU 105 个，库存口径已经影响经营判断。
目标：先把库存账实校准，不然补货和去化都会跑偏。
先查高频负库存按来源分三类排查未校正前先冻结深补
  • 先查高频负库存：先从销售高、库存低、还在建议补货清单里的负库存款开始校正，优先处理最影响营业额的货。
  • 按来源分三类排查：把负库存拆成盘点错误、调拨未入、销售未回写 3 类，店员照类别处理，不要混着查。
  • 未校正前先冻结深补：对负库存涉及的款先不要深补，只允许人工确认后补货，避免一边错一边继续压货。
跨季款先别补，先按季节处理
触发原因：当前有 246 个 SKU 被识别为跨季去化或暂缓补货，重点集中在 儿童袜、常规内裤、连裤袜 这些品类。
目标：避免把非主销季的货继续补深，把钱和货位留给当前季节。
有库存先去化没库存不再追补跨季款单独看板
  • 有库存先去化：如果 儿童袜、常规内裤、连裤袜 这些跨季品类手里还有库存，优先转到去化清单，用组合价、加价购、门口清货位来动销。
  • 没库存不再追补：如果某个跨季品类当前库存已经为 0，现阶段不要因为历史卖过就立即补，先等回到主销季再评估。
  • 跨季款单独看板：先看“跨季处理重点品类”，再下钻到单款。由老板每周决定这些品类是去化、暂缓还是等待下季，不跟当季补货混在一起。
会员占比高，适合做定向复购
触发原因：会员销售额占比 73.1%，会员已经是门店核心来源。
目标：把已有会员复购做深，而不是只靠新客自然进店。
换季提醒高客单老客回访复购带连带
  • 换季提醒：围绕 春季 到 夏季 的换季场景，给老客发“尺码 / 厚薄 / 到店试一轮”的提醒。
  • 高客单老客回访：优先回访 圆子、茵茵、呦米 这类高价值会员，话术聚焦“新到一批适合你家孩子的基础款”。
  • 复购带连带：会员到店不要只补单品，主推基础打底 + 袜品 + 家居这种顺手连带组合，放大客单价。

术语 Tips

鼠标悬浮或点按标签可看动作词解释；这里是完整词典。

经营销售额
只看正常商品销售，不包含道具，更接近门店真实营业额。
看法：先看这个数，再看客单价和订单数有没有一起增长。
客单价
平均每一单花了多少钱，公式是经营销售额 / 订单数。
看法：客单价低，优先查连带推荐和组合销售做得够不够。
库存覆盖天数
按最近销售速度估算，现有库存大约还能卖多少天。
看法：天数越高，压货越重；天数太低，容易断货。
负库存 SKU
系统里库存小于 0 的商品数量，说明账和货对不上。
看法：这个数不先处理，补货和去化判断都会失真。
建议补货 SKU
最近卖得不差、但库存偏低的商品数。
看法：先看销售金额高、库存接近 0、库存周数短的款。
建议去化 SKU
库存较高、近期卖得慢的商品数。
看法：先停补，再做搭配、组合或清货动作。
跨季处理 SKU
当前时间点不适合补货的商品数，包括跨季去化和暂缓补货。
看法：先看季节策略，冬款临近夏天不要直接补，优先去化或等待下季。
高压货品类
库存金额相对销售金额明显偏高的品类数量。
看法：高压货品类多，说明进货节奏和去化节奏都要收紧。
会员销售额占比
会员顾客贡献了多少经营销售额。
看法：占比越高，越值得做回访、复购提醒和老客经营。
道具参考
单独展示道具相关金额和库存，只作参考，不参与主经营判断。
看法：不要拿道具数值去判断真实商品卖得好不好。
毛利额
销售额减去进货成本后的余额，还没扣房租、工资、电费这些经营费用。
看法：毛利高不代表真正赚钱，还要继续看总费用和净利润。
净利润
毛利扣掉房租、工资、电费和其他月费用之后，真正剩下的钱。
看法：净利润为负时，重点不是继续压货，而是先提高毛利和控制费用。
保本销售额
按当前毛利率估算，本月至少卖到这个数，才够覆盖总费用。
看法：如果当前销售额还没过保本线，就先保主销、提客单、控库存。
