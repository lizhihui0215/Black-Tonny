小黑托昵门店经营入口

这是 GitHub Pages 的首页入口。老板和店员日常优先打开在线经营仪表盘；摘要、分析报告和 CSV 明细保留在下面，方便复盘和下载。

进入在线经营仪表盘 查看文字摘要 查看分析报告

老板怎么用

如果你只是想知道今天该做什么，直接打开仪表盘，不需要先看明细表。

  • 先看“今日经营重点”。
  • 再看“核心经营指标”和“库存风险”。
  • 最后才看补货机会、会员经营和详细数据。
