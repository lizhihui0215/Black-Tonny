这是 GitHub Pages 的首页入口。老板和店员日常优先打开在线经营仪表盘；如果要系统阅读经营、库存、培训和执行手册，直接进入文档中心。

本地一键抓取

这个按钮只在店里电脑上可用。它会调用本机服务，登录 Yeusoft 扫描关键报表，再重建最新仪表盘。GitHub Pages 本身不能直接抓数据，所以需要本地服务配合。

  • 先在店里电脑运行：python3 /Users/lizhihui/Workspace/black Tony/scripts/local_dashboard_service.py
  • 确认本地已填好：data/yeusoft_local_config.json 和 data/store_cost_snapshot.json
  • 然后回到这个首页，点击“本地一键抓取并刷新”。

老板怎么用

如果你只是想知道今天该做什么，直接打开仪表盘，不需要先看明细表。

  • 先看“今日经营重点”。
  • 再看“核心经营指标”和“库存风险”。
  • 最后才看补货机会、会员经营和详细数据。
