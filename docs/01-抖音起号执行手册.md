# 小黑托昵抖音起号执行手册

版本：v1.0  
更新日期：2026-03-11  
适用对象：单城市门店、老板娘本人出镜、目标为同城到店成交  
默认工作流：手机直拍 + 剪映剪辑 + 每周 1 次轻直播

## 使用说明

- 先把全文中的占位符替换掉：`[城市]`、`[门店地址]`、`[老板娘称呼]`、`[营业时间]`、`[门店电话/微信]`。
- 先执行第 2 章首页配置，再按第 4 章连续执行 14 天。
- 视频内容只围绕 `儿童贴身衣物怎么选`，不要扩成泛童装号，也不要拍成纯门店 vlog。
- 账号记忆顺序固定为：`先记住老板娘` -> `再记住小黑托昵 / Black Tonny` -> `最后到店成交`。

## 1. 账号定位与人设

### 1.1 核心目标

- 目标不是做全国卖货号，而是让本地宝妈在刷到你后，愿意评论、私信、到店。
- 账号唯一成交路径固定为：`同城看到视频` -> `评论/私信问问题` -> `到店试穿/选码` -> `线下成交`。
- 账号所有内容都服务于一个判断：`这位老板娘是不是能帮我省掉给孩子挑贴身衣物的时间和试错成本`。

### 1.2 目标人群

- 核心用户：本地正在给孩子买内衣、打底、秋衣秋裤、袜品、居家贴身衣物的宝妈。
- 高频问题：`厚薄怎么选`、`尺码怎么定`、`换季要不要买`、`孩子怕热/怕冷怎么办`、`同价位先看什么`。
- 用户需要的不是品牌故事，而是 `一个门店里每天都在帮别人选的人` 给出的稳定建议。

### 1.3 人设定义

- 人设名称：`[城市]小黑托昵老板娘`
- 人设定位：`在门店里替宝妈先把儿童贴身衣物筛一遍的人`
- 人设边界：
  - 不是品牌官方发言人
  - 不是检测机构或医学专家
  - 不是夸张种草主播
  - 是一线接待、选码、试穿建议的实际执行者

### 1.4 统一表达

- 统一口头句式：
  - `我是[城市]小黑托昵老板娘，平时在店里最常帮宝妈解决的就是这类问题。`
  - `我在门店里通常会先看这 3 个点。`
  - `这个场景下更适合这样选，不一定要买最厚的。`
  - `如果你家孩子是这个年龄段，我建议先按这个逻辑来。`
- 统一内容原则：
  - 多讲 `怎么选`
  - 少讲 `我们多厉害`
  - 多拍 `真实接待现场`
  - 少拍 `空镜头堆砌`

### 1.5 不能偏航的 5 条红线

- 不做泛童装穿搭号，主线只讲儿童贴身衣物。
- 不做官方品牌宣传片，不拍大段 logo、招商感画面。
- 不讲无法证明的功能性卖点，不用绝对化措辞。
- 不做低价叫卖号，活动和价格只放结尾或评论区轻带。
- 不多人轮流出镜，长期只保留一个固定主讲人。

## 2. 首页配置与置顶内容

### 2.1 昵称模板

昵称必须满足：`[城市] + 老板娘/店长称呼 + 小黑托昵/儿童贴身棉品`

推荐直接选以下 3 个模板中的一个：

1. `[城市]小黑托昵老板娘`
2. `[城市]店长说儿童贴身棉品`
3. `[城市]小黑托昵儿童贴身棉品老板娘`

默认最推荐：`[城市]小黑托昵老板娘`  
原因：最容易记、最像真人账号、最适合同城成交。

### 2.2 简介模板

默认简介：

> 我是[城市]小黑托昵老板娘，平时在店里帮宝妈把儿童贴身衣物先筛一遍，省你选面料和尺码的时间。

备选简介：

> 在[城市]做儿童贴身棉品门店这些年，平时最常帮宝妈解决厚薄、尺码和换季怎么选。

### 2.3 头像、背景和主页观感

- 头像：老板娘本人半身照，背景尽量在门店货架前，不要用纯品牌 logo。
- 背景图 / 头图文案：`帮宝妈先把儿童贴身衣物筛一遍`
- 主页固定露出顺序：
  - 门店现场
  - 摸面料
  - 选尺码
  - 试穿区
  - 顾客问题
  - 货架补货

### 2.4 主页置顶 3 条

#### 置顶 1：我是谁

- 标题：`我是怎么在店里帮宝妈选儿童贴身衣物的`
- 目的：让新用户 10 秒知道你是谁、做什么、在哪。
- 镜头顺序：
  1. 门头或货架 1 秒
  2. 老板娘正面出镜 4 秒
  3. 摸面料 / 整理货架 4 秒
  4. 顾客挑码现场 4 秒
  5. 到店 CTA 3 秒
- 口播模板：
  - `我是[城市]小黑托昵老板娘。`
  - `平时在店里，我最常帮宝妈先把儿童贴身衣物筛一遍。`
  - `比如换季厚薄怎么选、孩子多大穿多大、怕热怕冷怎么搭。`
  - `你在[城市]，又刚好也在纠结这类问题，可以来店里我帮你先看。`

#### 置顶 2：换季怎么选儿童贴身衣物

- 标题：`换季别急着给孩子买最厚的，先看这 3 个点`
- 目的：树立决策价值，而不是单纯卖货。
- 镜头顺序：
  1. 不同厚薄款摆桌面
  2. 手摸面料特写
  3. 按年龄段或怕热怕冷做对比
  4. 一句话结论
- 结尾 CTA：`你家孩子现在多大，评论区告诉我，我按门店经验给你说。`

#### 置顶 3：来店里能解决什么问题

- 标题：`来店里不是为了看我直播，是为了把这几件事一次解决`
- 目的：提前说明到店价值，减少用户对“到店会不会白跑”的顾虑。
- 必讲 3 个点：
  - 帮你判断厚薄
  - 帮你缩小尺码范围
  - 能现场摸、比、试、问
- 结尾 CTA：`你离[门店地址]近的话，来之前私信我孩子年龄段，我先帮你备一轮。`

## 3. 内容系统与栏目比例

### 3.1 内容比例

| 类型 | 比例 | 目标 | 典型场景 | 容易犯错 |
| --- | --- | --- | --- | --- |
| 真实门店日常 | 50% | 让人相信你真的每天都在解决问题 | 开店、配货、上新、接待、试穿、老客复购 | 拍成流水 vlog，没有结论 |
| 宝妈决策内容 | 30% | 建立“会选”的信任 | 换季怎么选、厚薄对比、不同年龄段怎么挑 | 讲概念太空，不落到门店实物 |
| 到店证明 / 轻转化 | 20% | 让用户愿意留言和到店 | 试穿反馈、老客回购、到店提问、服务细节 | 只喊优惠，没有信任铺垫 |

### 3.2 单条视频固定结构

每一条都按这个结构拍：

1. 前 3 秒抛真实问题
2. 店内现场或实物演示
3. 一句话给结论
4. 引导同城宝妈评论或私信

标准模板：

> 开头：`很多宝妈一到换季就先给孩子买最厚的，这个在店里我基本都会先劝一下。`  
> 中段：展示 2-3 款不同厚薄的实物，边摸边讲判断逻辑。  
> 结尾：`如果你家孩子是[年龄段]，先别急着买，评论区告诉我情况，我按门店经验帮你缩一下范围。`

### 3.3 题型池

固定只拍以下 8 类，不再额外开新坑：

1. 老板娘真实接待
2. 换季选品建议
3. 真实问题答复
4. 尺码判断
5. 老客复购
6. 上新 / 配货 / 整理货架
7. 到店试穿证明
8. 直播答疑切片

### 3.4 每天顺手拍的素材池

- 开门、开灯、整理货架
- 手摸面料特写
- 同款不同厚薄摆在一起
- 拿尺码对比两件衣物
- 顾客背影选货
- 试穿区整理
- 老客复购打包
- 收银台问答
- 门店门头
- 货架局部、价格签局部、标签局部

规则：每天至少拍 10 段 5-8 秒的短素材，后面剪视频不用再临时补拍。

### 3.5 周节奏

- 第 1 周：覆盖 `真实接待`、`换季建议`、`问题答复` 三大类。
- 第 2 周：把优胜题型放大做成系列。
- 如果第 1 周还看不出绝对优胜，默认放大这 2 类：
  - `换季选品建议`
  - `尺码判断`

## 5. 短视频脚本模板与开场库

### 5.1 固定拍法

一条门店短视频默认 20-35 秒，镜头数量控制在 5-8 个。

基础顺序：

1. 问题开头
2. 门店现场
3. 产品或尺码演示
4. 一句话结论
5. 评论 / 私信 CTA

### 5.2 脚本模板 1：老板娘真实接待

- 开头：`店里今天又有宝妈一进来就问我这个问题。`
- 镜头：
  1. 老板娘在门口或货架前
  2. 顾客背影进店
  3. 货架取货
  4. 尺码展示
  5. 一句话结论
- 口播骨架：
  - `今天一位宝妈先问我，孩子现在到底该买多厚。`
  - `我一般不会先推荐最厚的，我会先看孩子日常活动量。`
  - `再看她是上学通勤，还是只是小区活动。`
  - `这个场景下，我会先从中间厚度开始缩。`
  - `你家孩子如果也在这个阶段，评论区告诉我年龄段。`

### 5.3 脚本模板 2：换季选品建议

- 开头：`换季最容易买错的，不是花色，是厚薄。`
- 镜头：
  1. 三件不同厚薄平铺
  2. 手摸面料
  3. 厚薄折叠对比
  4. 老板娘一句话下结论
- 口播骨架：
  - `很多宝妈会下意识给孩子买最厚的。`
  - `但门店里我通常先看体感、活动量、穿着场景。`
  - `如果是日常通勤，先从中间厚度更稳。`
  - `孩子特别怕热或活动量大，再往薄的看。`
  - `你家孩子是哪种情况，评论区告诉我。`

### 5.4 脚本模板 3：尺码判断

- 开头：`儿童贴身衣物最容易买错的，其实是尺码。`
- 镜头：
  1. 两个尺码平铺
  2. 标签特写
  3. 袖长/裤长对比
  4. 一句话判断
- 口播骨架：
  - `很多人先看年龄，但我在店里不会只看这个。`
  - `我会先看身高，再看活动空间。`
  - `贴身衣物太绷和太空都不舒服。`
  - `先把范围缩到 2 个码，再来店里试，最省时间。`
  - `私信我孩子身高体重，我先帮你缩一轮。`

### 5.5 脚本模板 4：真实问题答复

- 开头：`评论区这个问题，这两天被问了很多次。`
- 镜头：
  1. 评论截图打码
  2. 老板娘口播
  3. 货架拿货对比
  4. 结论
- 口播骨架：
  - `有人问我，孩子爱出汗是不是就只买最薄的。`
  - `这个不能一刀切。`
  - `还要看上学、睡觉、活动量和温差。`
  - `我在店里更倾向先按场景选，再细调厚薄。`
  - `你把孩子情况发我，我按场景给你说。`

### 5.6 脚本模板 5：老客复购 / 到店证明

- 开头：`老客回购最能说明问题，但不是说明便宜。`
- 镜头：
  1. 复购打包
  2. 老客背影
  3. 老板娘整理同款
  4. 结尾 CTA
- 口播骨架：
  - `今天这个妈妈又回来拿上次那种。`
  - `她看中的不是我说得多好，是省心。`
  - `知道来店里能一次把厚薄和尺码一起缩掉。`
  - `如果你也不想反复试错，可以先私信我孩子情况。`

## 6. 直播 SOP

### 6.1 直播定位

- 直播目标：答疑、选码、换季建议、门店信任。
- 直播禁忌：不要持续喊价格、不要做“321 上链接”、不要讲夸张承诺。
- 直播时长：30-60 分钟即可，不追求长。

### 6.2 直播前准备

- 手机支架 1 个
- 补光灯 1 个
- 代表款 6-10 件
- 纸笔或备忘录，记录高频问题
- 评论区固定欢迎语 1 条
- 今日直播主题标题 1 条

### 6.3 直播流程

#### 第 1 段：开场 3 分钟

固定开场：

> 我是[城市]小黑托昵老板娘，今天这场只帮大家解决 3 类问题：厚薄怎么选、尺码怎么缩、最近适不适合来店。你直接把孩子年龄段打在公屏，我按门店经验给你说。

#### 第 2 段：集中答疑 10-15 分钟

- 先答评论区最高频 3 个问题
- 每个问题都要拿实物讲，不空口讲概念
- 每讲完一个问题都加一句：
  - `你家孩子如果是这个情况，我建议你先按这个思路来。`

#### 第 3 段：实物演示 10-15 分钟

- 演示 3 类代表款：
  - 偏薄
  - 中间厚度
  - 稍厚
- 每类只讲：
  - 适合什么场景
  - 哪类孩子更常选
  - 来店时怎么快速判断

#### 第 4 段：同城承接 5-10 分钟

固定收口：

> 你在[城市]的话，来店前可以先把年龄段和大概身高发我，我先帮你把范围缩一轮，省得你白跑。

### 6.4 直播后的动作

- 直播结束后 30 分钟内，整理 3 类高频问题
- 从直播里剪 1-2 条问答切片，作为次日视频素材
- 把问得最多的 1 个问题列入下一天选题

## 7. 评论区 / 私信 / 到店转化 SOP

### 7.1 只承接 3 类高价值问题

1. 门店地址
2. 尺码 / 年龄段
3. 最近适不适合来店

### 7.2 评论区标准回复

#### 场景 1：有人问门店在哪

回复模板：

> 我在 `[城市][门店地址]`，你来之前可以先把孩子年龄段发我，我先帮你看一下适合从哪类开始试。

#### 场景 2：有人问孩子几岁穿什么码

回复模板：

> 贴身衣物我一般不只按年龄看，你把身高和大概体型私信我，我先帮你把范围缩到 1-2 个码。

#### 场景 3：有人问最近适不适合买

回复模板：

> 这个要看你家孩子最近通勤和活动场景，如果你在 `[城市]`，把年龄段和最近穿着情况发我，我按门店经验先帮你判断是不是现在就该换。

### 7.3 私信承接流程

固定四步：

1. 先问孩子年龄段 / 身高
2. 再问最近场景：上学、居家、活动量
3. 给一个初步范围，不一次说死
4. 最后引导到店试穿或现场比较

私信模板：

> 先别急着定具体哪一件。你把孩子年龄、身高、最近是上学通勤还是居家为主告诉我，我先帮你缩一下厚薄和尺码范围。

### 7.4 到店承接流程

- 到店第一句建议：
  - `你是从抖音过来的吗？`
- 如果对方说是：
  - `那我先按你在评论区说的情况，带你从这两类开始看。`
- 到店重点不是介绍品牌，而是：
  - 先判断厚薄
  - 再缩尺码
  - 最后再谈具体款式

### 7.5 必做记录

每天关店前记录 3 项：

1. 今天有多少人问地址
2. 今天有多少人问尺码 / 年龄段
3. 今天有多少人到店时主动提到“刷到抖音来的”

## 8. 风险避雷与合规话术

### 8.1 禁用表达

- `最安全`
- `绝对放心`
- `全网最好`
- `完全不会出问题`
- `所有孩子都适合`
- `闭眼买`
- `最顶级`
- `比谁谁谁都好`

### 8.2 替代表达

| 不建议说 | 建议改成 |
| --- | --- |
| 最安全 | 这类孩子穿着反馈更稳定 |
| 绝对放心 | 这个场景下更适合 |
| 全网最好 | 这是门店里宝妈问得比较多的一类 |
| 闭眼买 | 如果你家孩子是这个情况，可以优先从这类看 |
| 所有人都适合 | 这个年龄段或这个场景更常用 |
| 一定要买厚的 | 我在店里通常会先看体感和场景 |

### 8.3 内容边界

- 不主动做品牌官方背书。
- 不谈自己没有掌握证据的检测、质量结论或功能承诺。
- 不把儿童贴身衣物说成保健品或医疗用品。
- 不把个别顾客反馈说成普遍结论。
- 不引导用户相信“线上看视频就能完全替代试穿 / 到店判断”。

### 8.4 遇到敏感提问时的统一回复

#### 评论区有人追问品牌历史、旧闻或质量争议

统一回复：

> 我这边主要分享的是门店里真实的选购和试穿经验，涉及品牌历史或官方材料的问题，建议还是以官方公开信息和最新资料为准。如果你是想解决孩子当前穿着怎么选，我可以先按门店经验帮你缩范围。

#### 用户追问“你能不能保证一定怎样”

统一回复：

> 这种我不建议说太满。贴身衣物还是得结合孩子年龄、体感和实际场景来选，我可以先按门店里常见情况给你建议。

### 8.5 顾客隐私与拍摄边界

- 顾客正脸默认不入镜，除非得到明确同意。
- 儿童试穿区只拍局部、背影、产品和老板娘手部动作。
- 评论截图、私信截图统一打码。

## 9. AI 工具完整工作流

### 9.1 使用原则

- AI 负责提效，不负责替你冒充真实门店经验。
- AI 只用于：选题、脚本、提纲、封面灵感、剪辑提效、复盘整理。
- 所有成片内容最后都要回到真实门店、真实出镜、真实口播。

### 9.2 环节一：选题与脚本

**环节**：选题与脚本

优点：

缺点：

缺点：
- 官方说明里也提示内容可能不准确，不能把它生成的话直接照搬做事实陈述。

链接：[腾讯元宝](https://yuanbao.tencent.com/app)

- 适合拿评论区问题临时生成直播提纲或回复草稿。

缺点：
- 更适合临时问答，不如 Kimi 适合做成体系脚本库。

**最推荐**：Kimi  
**适用场景**：14 天排期、系列脚本、每周复盘表、直播提纲

- 轻量复盘比较快。
缺点：
- 对长期表格化管理不如 Kimi Sheets 直观。
- 字段一致性要靠你自己盯。

**方案 3：腾讯元宝**  
链接：[腾讯元宝](https://yuanbao.tencent.com/AI)  
优点：
- 适合快速生成直播提纲、答疑要点和评论回复草稿。
- 对当天临时爆出来的问题反应快。
- 不需要复杂设置。
缺点：
- 更像轻量助手，不是最适合长期复盘系统的工具。
- 数据结构化能力不够强时，后续回看不方便。

**最推荐**：Kimi Sheets / Docs  
**适用场景**：每周复盘、下周排期、爆款问题汇总  
**预算备注**：先做 1 张固定复盘表，用起来后再决定是否加其他工具  
**替代方案**：如果你更习惯先把截图堆在一起处理，就先用豆包云盘

推荐提示词：

```text
请把这一周的抖音评论和私信问题，按“门店地址”“尺码/年龄段”“最近适不适合来店”“其他”四类整理成表格，并告诉我下周最应该继续拍的两个系列。优先保留能带来咨询的问题，不要只按播放量判断。
```

## 10. 每周复盘表与淘汰规则

### 10.1 只看 4 个指标

1. 同城播放占比
2. 宝妈咨询数
3. 到店时提到抖音的人数
4. 哪类开场最留人

### 10.2 每周复盘表

| 视频编号 | 题型 | 同城播放占比 | 咨询数 | 强购买问题是否出现 | 到店提及抖音人数 | 开场留人感受 | 结论 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 视频 A | 例如换季选品 |  |  |  |  |  | 保留 / 观察 / 淘汰 |
| 视频 B | 例如真实接待 |  |  |  |  |  | 保留 / 观察 / 淘汰 |
| 视频 C | 例如尺码判断 |  |  |  |  |  | 保留 / 观察 / 淘汰 |

### 10.3 保留 / 观察 / 淘汰规则

#### 保留

满足任一条就保留：

- 评论区出现 `尺码、年龄、地址、到店时间` 这类强购买问题
- 私信明显增加
- 有用户到店时主动提 `刷到你视频来的`

#### 观察

- 播放还可以，但咨询不明显
- 互动多是泛评论，如 `好看`、`支持一下`，没有真实购买问题

#### 淘汰

- 连续 2 条只有播放没有咨询
- 用户看完记不住你是谁，也不知道来店能解决什么
- 内容拍成纯 vlog，看完没有明确结论

### 10.4 下周迭代规则

- 留下最能带咨询的 2 类题型，连发 3-5 条
- 每周至少保留 1 条到店证明内容
- 如果某类视频只有播放没有咨询，直接降频
- 如果评论区某个问题重复出现 3 次以上，下周必须做成单独视频

- 今天这条视频有没有明确问题开头
- 有没有门店现场，而不是纯口播
