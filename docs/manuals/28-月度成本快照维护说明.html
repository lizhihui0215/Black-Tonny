月度成本快照维护说明¶

这份说明对应本地 月度成本维护台，目标是让老板每个月维护一次成本快照，不需要手改 JSON 文件。

入口¶

  • Pages 首页本地入口：/costs/
  • 本地文件入口：docs/costs/index.html

使用前提¶

  1. 在店里电脑运行本地服务： python3 /Users/lizhihui/Workspace/black Tony/scripts/local_dashboard_service.py
  2. 浏览器打开： docs/costs/index.html
  3. 页面会自动读取本地： data/store_cost_snapshot.json

页面能做什么¶

  • 读取当前成本快照
  • 修改本月销售额和进货成本
  • 修改固定费用明细
  • 按当前规则自动计算：
  • 毛利额
  • 毛利率
  • 固定费用合计
  • 人工费用
  • 总费用
  • 净利润
  • 保存回本地 store_cost_snapshot.json
  • 可继续一键触发仪表盘重建

当前人工口径¶

  • 员工工资当前按固定 7,500 计算
  • 不参考截图里的逐人工资拆分
  • 如果后面人工口径变了，再统一调整

维护建议¶

  • 每个月至少更新 1 次
  • 如果租金、电费、推广费发生明显变化，也应同步更新
  • 更新完成本后，建议立即重建仪表盘，保证利润和保本判断同步刷新
