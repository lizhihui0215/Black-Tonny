# 小黑托昵老板周复盘与执行校准

版本：v1.0  
更新日期：2026-03-12  
适用对象：老板 / 老板娘、2 位员工  
配套文档：
- `docs/07-店铺日常经营复盘表.md`
- `docs/08-小团队门店分工与排班手册.md`
- `docs/09-员工培训与带教SOP.md`

## 使用说明

- 这份文档解决的是 `老板每周怎么带着 2 位员工复盘、对齐和定动作`。
- 小团队门店不需要太正式的会议感，重点是把经营问题、执行问题、培训问题一次说清楚。
- 默认每周固定复盘 1 次，时间控制在 20-30 分钟。

## 1. 老板周复盘目标

每周复盘先只解决这 4 件事：

1. 这周经营有没有稳住
2. 这周执行有没有掉链子
3. 这周培训有没有真正起作用
4. 下周最该抓哪 2-3 件事

## 2. 参与人

建议固定是这 3 个人：

1. 老板 / 老板娘
2. 员工 A
3. 员工 B

## 3. 固定议程

### 3.1 先看经营结果

固定看这 6 个：

1. 总到店人数
2. 总成交人数
3. 抖音到店人数
4. 老客复购人数
5. 最常被问的 3 个问题
6. 最带咨询的内容题型

### 3.2 再看人员执行

固定看这 5 个：

1. 这周哪类顾客最容易成交
2. 哪位员工在哪个接待环节最容易跑偏
3. 哪个时段最容易顾不过来
4. 哪个交接信息最容易漏
5. 下周哪位员工最需要复训哪一项

### 3.3 再看培训结果

固定看这 4 个：

1. 新员工有没有进入正常接待
2. 接待开口有没有统一
3. 抖音到店顾客有没有被正确承接
4. 哪个问题需要下周集中再练

### 3.4 最后定下周动作

每周只定：

1. `1 个经营重点`
2. `1 个执行重点`
3. `1 个培训重点`

不要一周定太多，不然执行会散。

## 4. 小团队怎么做表现复盘

这份文档不建议一开始做复杂绩效表，先做最小版复盘。

### 4.1 先看 4 类表现

1. `接待表现`
   是否能自然接顾客、是否会缩小范围
2. `成交承接表现`
   是否会按厚薄、尺码、具体款的顺序推进
3. `记录反馈表现`
   是否会记高频问题、抖音到店、老客信息
4. `团队配合表现`
   是否会交接、补位、配合现场安排

### 4.2 每周判断方式

先不要复杂打分，先分成：

1. `稳定`
2. `需要提醒`
3. `需要复训`

## 5. 周复盘记录模板

每周至少记录下面这几项：

| 周次 | 经营重点结论 | 执行问题 | 培训问题 | 下周经营重点 | 下周执行重点 | 下周培训重点 |
| --- | --- | --- | --- | --- | --- | --- |
| 例如 第 1 周 |  |  |  |  |  |  |

## 6. 周复盘结论模板

每周复盘结束时，用这段话收口：

> 本周门店经营最需要继续放大的是 `[经营重点]`，执行上最需要调整的是 `[执行问题]`，培训上最需要补的是 `[培训问题]`。下周我们只先抓 `[重点1]`、`[重点2]`、`[重点3]`。

## 7. 哪些问题必须进每周复盘

下面这些问题不要只私下说，要进老板周复盘统一处理：

1. 抖音到店顾客总是接不住
2. 某个时段反复出接待问题
3. 新员工上手太慢
4. 接待话术越来越散
5. 店里记录总是断

## 8. 当前最先开始的周复盘做法

如果你现在还没正式跑过老板周复盘，先按这个最小版本来：

1. 每周固定一个时间
2. 先看 `经营数字 + 高频问题`
3. 再看 `哪位员工需要复训`
4. 最后只定下周 3 个动作

先把复盘跑起来，再慢慢补细节，效果会比一直靠临时沟通强很多。
