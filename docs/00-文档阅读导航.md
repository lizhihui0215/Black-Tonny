# 小黑托昵文档阅读导航

版本：v1.1  
更新日期：2026-03-13  
适用对象：老板 / 老板娘、员工、门店协作人员

## 使用说明

- 这份导航解决的是 `现在文档越来越多，先看哪份、按什么顺序看`。
- 默认不要从头到尾全部读，先按你的目标找入口。

## 8. 脑图和 XMind 怎么看

- `mindmaps/*.md`
  这是可继续改的 Markdown 脑图源文件
- `mindmaps/*.xmind`
  这是可以直接用 XMind 打开的版本

建议：

1. 平时改内容，优先改 `docs/` 和 `mindmaps/*.md`
2. 需要演示、开会、培训时，优先打开 `.xmind`
